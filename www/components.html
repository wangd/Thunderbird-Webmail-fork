Component Manual
  1. Mode

The Hotmail extension has three methods of communication with Microsoft's servers ScreenScraper, WebDav, and Beta. The screen scraper method parses the web page for the necessary information. Weddav is the same method Outlook and Outlook Express use to communication with Microsoft's. Beta is used to communicate with the Beta Hotmail site.

The Webdav method is faster and more reliable than the screen ripper method but is restricted of old free account or paid accounts.


  1. POP

  • Empty Trash

Checking this option will cause the server trashcan to be emptied after every email check.


  • Download Junk Mail

Email will be download from the Inbox and the Junkmail folder.


  • Download Unread Mail Only

Only emails marked as unread on the server will be download. Once an email has been read Thunderbird WILL NOT be able access the server copy again.


  1. SMTP

  • Save Copy In Sent Items

A copy will be save in the servers Sent Items folder.


  • Multipart

A rich text email contains two parts a plain text and html part.
