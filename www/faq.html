FAQs

How do I install these extensions

  1. Right-Click the extension link and choose "Save Link As..." to Download and save the file to your hard disk.
  2. In Mozilla Thunderbird, open the extension manager (Tools Menu/Add-ons).
  3. Click the Install button, and locate/select the file you downloaded and click "OK".

Why is the extension failing to download emails with large attachments.

Thunderbird is timing out before the extension has downloaded the email. To increase the timeout delay Open Thunderbird's option panel, select Advanced ->Network And Disc Space, now increase the value of connection timeout..

Why am I get the error "Sending of username did not succeed. Mailserver localhost responded: undefined is a unsupported domain"

The username you entered in to Thunderbird when creating the account is missing a domain (ie @Hotmail.com, @Yahoo.co.uk)

The Webmail server status is "ERROR"

Check your firewall isn't blocking Thunderbird from opening a connection to localhost.


Some OS's block ports below 1024, set the ports used by Webmail to values higher then 1024
