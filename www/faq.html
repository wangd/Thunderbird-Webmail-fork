FAQ

Yahoo / Hotmail extension fails to download emails with large attachments.
Thunderbird is timing out before the extension has downloaded the email.  Thunderbird's POP timeout defaults to 45 seconds,  to increase this time you need to add this "user_pref("mail.pop3_response_timeout", 180);" to the prefs.js (the number is seconds).



Extension fails to donwload any emails.
This extension require Thunderbird's cookie handling to be in it’s default state (ie disabled). To return the cookie state back to it’s default place the following in the prefs.js "user_pref("network.cookie.cookieBehavior", 2);". You will also need to delete the cookie.txt file in your profile.
