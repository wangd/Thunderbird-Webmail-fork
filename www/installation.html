Installation

This extension is for Thunderbird v1 only. Thunderbird’s cookie handling must be in it’s default state (ie disabled).

The WebMail extension must be installed first and Thunderbird restarted before you install other components

You will need at least two of the following extensions

Web-Mail-0.3.3 Required
Web-Mail-0.3.3
Web-Mail-0.3.3
Hotmail-0.4.1 Optional
Yahoo-0.2.14 Optional
Lycos-0.1.3 Optional
Lycos-0.1.3
