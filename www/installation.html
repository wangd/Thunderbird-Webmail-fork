This extension is for Thunderbird version 1.0.0 to 1.0.6 only. Thunderbird’s cookie handling must be in it’s default state (ie disabled).

The WebMail extension must be installed first and Thunderbird restarted before you install other components

You will need at least two of the following extensions

Web-Mail-0.3.4 Required
Hotmail-0.4.3 Optional
Yahoo-0.2.16 Optional
Lycos-0.1.5 Optional
MailDotCom-0.1.3 Optional
