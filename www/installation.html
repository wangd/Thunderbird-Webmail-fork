Download

The extensions listed below are for Thunderbird 1.5.

You will need at least two of the following.

The Webmail extenesion must be installed first and Thunderbird restarted before you can install the other components.

Web-Mail-1.0.11 Required
Hotmail-1.0.5 Optional
Yahoo-1.1.1 Optional
Lycos-1.0.0 Optional
MailDotCom-1.0.1 Optional
Gmail-0.5.3 Optional
Libero-0.9.1 Optional
AOL-1.0.0 Optional

How to Install

  1. Right-Click the link above and choose "Save Link As..." to Download and save the file to your hard disk.
  2. In Mozilla Thunderbird, open the extension manager (Tools Menu/Extensions).
  3. Click the Install button, and locate/select the file you downloaded and click "OK".
