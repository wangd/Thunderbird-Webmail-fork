Account Setup
Setup

I’ll assume you have installed the WebMail extension and at least one of the optional components, if you haven't please do so.
  1. The first step is to check the WebMail extension is running. Go to the WebMail options, on the serve panel the status of the three server is displayed. The status for the enabled server should be green. If the WebMail extension fails to start this could be due to the operating system blocking ports below 1024, try setting the port higher than 1024 and restart Thunderbird.

  2. Check you have installed the necessary domain handlers. Go to WebMail options, on the domains panel is a list of supported domains.

You are now ready too set-up an email account in Thunderbird. In the following example I'll be using a fictitious Hotmail account (BillyG@hotmail.com).

POP
  1. From the main window menu bar select File->New->Account.

  2. In the Account wizard select “Email Account”

  3. On the Identity page set your name and email address.
    Ex
    Your Name : Bill Gates
    Email Address : BillGates@hotmail.com

  4. For the server information use the following settings
    Server Type : POP
    Incoming Server : localhost

  5. The user name must include the domain
    Ex
    Incoming UserName : BillGates@Hotmail.com
    Outgoing UserName : BillGates@Hotmail.com

  6. Follow the new account wizard to the end

  7. If you changed the port number during the WebMail setup stage you must change the port number for this account in the Account Settings dialog box.
    Ex
    Tools->Account Settings->Bill Gates->Server Settings

SMTP
  1. Open “Account settings”, select the “Outgoing server” option from the tree and press the “Add” button.

  2. You must set up a SMTP account for each POP account.
    Ex
    Server Name : localhost
    User Name : username@domain
